# CV Portfolio

Welcome to **CV Portfolio**! This project is a modern web application built with React and Vite, designed to create and showcase an interactive professional portfolio—perfect for developers, designers, and anyone who wants to highlight their experience and skills.

## 🚀 Main Features

- **Responsive and modern design**: Attractive interface adaptable to any device.
- **Customizable sections**: About, Experience, Education, Projects, Skills, and Contact.
- **Reusable components**: Component-based architecture for easy maintenance and scalability.
- **Interactive charts**: Visualize skills with radar charts.
- **Animations and visual effects**: Enhanced user experience with subtle animations.
- **Easy deployment**: Ready to be published on platforms like Vercel, Netlify, or GitHub Pages.
- **Integrated testing**: Unit tests with Jest and React Testing Library.

## 🛠️ Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/cv-portfolio.git
   cd cv-portfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open in your browser:**
   Visit [http://localhost:5173](http://localhost:5173)

## 🧪 Run tests

```bash
npm test
```

## 📦 Build for production

```bash
npm run build
```

## ✨ Customization

- Edit files in `src/components/sections/` to update each section's information.
- Change styles in `src/styles/` to adapt the appearance to your liking.
- Add your images in `public/images/`.
